# react-boilerplate

This is a pure React client-side app (SPA) boilerplate.

## Features

- React 16.x
- Webpack 4.x
- Babel 7.x
- ESLint 6.x
- React Router 5.x
- Material UI 4.x
- Prettier setup

## Install

```
npm i
```

## Run for dev

```
npm run dev
```

## Build for production

```
npm run build
```
